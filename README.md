# hospital
基于Springboot Vue医院挂号系统
## 一、 项目介绍
       基于SpringBoot Vue医院管理系统

       角色：病人、医生、管理员

       整体功能包含医生信息管理，患者信息管理，挂号信息列表，医药信息管理，检查项目管理，病床信息管理，值班安排信息，数据分析，预约挂号，我的挂号，我的住院，个人信息，当天挂号，我的挂号，申请当天入院，个人信息管理。
